The Seventh Layer of the OSI Model, also known as the Application Layer

Definition + Role

Top layer that acts as the interface between the user applications and the network

Function

Provides network services directly to the user and facilitates commmunication protocols for tasks like web browsing, email, and file transfer

Important for user experience and security as it is the direct interface for users to interact with the software, and is where security measures are implemented (i.e. passwords.. two factor authentication)

The Fifth Layer of the OSI Model, also known as the Session Layer

Definition + Role

Establishes, manages, and terminates sessions between communicating application processes

Handles session checkpointing, synchronization, recovery, and ensuring data exchange occurs in an organized manner
